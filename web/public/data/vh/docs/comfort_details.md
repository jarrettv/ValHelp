# Comfort Details

Per-comfort-item notes that appear on the detail screen after the recipe. Each section is keyed to an item by an HTML comment containing the item code on the `###` heading.

Format:

    ### Stone Bed <!-- BedStone -->
    Notes here. Supports **bold**, *italic*, bullet lists, and tables.

`##` category headers are for author organization — they are not rendered on the detail screen. Only `### Title <!-- Code -->` sections are extracted.
