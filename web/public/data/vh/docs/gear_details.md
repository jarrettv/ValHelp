# Gear Details

Per-gear notes that appear on the detail screen between the recipe and the properties. Each section is keyed to an item by an HTML comment containing the item code on the `###` heading.

Format:

    ### Iron Helmet <!-- HelmetIron -->
    Notes here. Supports **bold**, *italic*, bullet lists, and tables.

`##` category headers are for author organization — they are not rendered on the detail screen. Only `### Title <!-- Code -->` sections are extracted.
